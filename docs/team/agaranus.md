# Louis Agara Perin - Project Portfolio Page

## Overview

Dextro is a desktop application for managing student academic records, optimised for use via a Command Line Interface (CLI) while still having the benefits of a Graphical User Interface. It helps educators and administrators track student progress, manage course modules, and monitor academic performance efficiently.
